# PokéIdle Trainer

**PokéIdle Trainer** est un jeu fan-made idle inspiré de l’univers Pokémon.


## Installation locale

1. Clonez ou téléchargez ce dépôt.
2. Ouvrez **index.html** dans votre navigateur.
